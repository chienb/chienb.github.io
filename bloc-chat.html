Bloc Chat

Bloc Bootcamp

2016

Real-time chat application project for Bloc's Frontend Web Development Track

Bloc Chat is built using the AngularJS framework. Together with FireBase NoSQL DB and AngularFire, Chat Room commits messages to the backend and returns a promise in real time with two way data binding. This simple, yet elegant application could be a standalone tool or integrated into a larger project for communication purposes.
